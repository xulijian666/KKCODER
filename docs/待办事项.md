# KKCODER 远程访问 - 待办事项

## 已完成（本次开发）

### 桌面端 Rust 后端
- [x] axum WebSocket 服务器（端口 9527），支持远程终端输入输出
- [x] PIN 配对认证（6 位 PIN，5 分钟过期，UUID v4 token）
- [x] 断线重连（seq + ring buffer + replay）
- [x] frp 客户端集成（启动/停止/状态，支持 TLS）
- [x] 新增 12 个 Tauri 命令（远程配置、设备管理、frp 控制）
- [x] SessionRegistry 注册修复：PtyManager 创建时注入 registry，桌面端打开的会话正确标记为 active
- [x] paired_devices DashMap key 修复：用 token 做 key 而非 device_id，重启后 token 不丢失
- [x] PIN 存储修复：generate_pairing_pin 和 verify_pairing 都走数据库，不再对不上
- [x] 测试 URL 修复：自动补全 http:// 前缀
- [x] 会话列表 API：从 SQLite 读取，返回 active 字段，支持手机端按项目分组
- [x] 创建会话 API：POST /api/sessions，手机端可创建新会话
- [x] 启动会话 API：POST /api/sessions/:id/spawn，加入 spawn 队列，桌面端轮询执行
- [x] 监听模式：localhost（仅本机）/ all（所有设备）可选
- [x] 公网访问地址：存储 + 连通性测试
