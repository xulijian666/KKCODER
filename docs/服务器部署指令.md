# KKCODER 远程访问 - 服务器部署指令

## 这是什么

我做了一个桌面应用 **KKCODER**，在电脑上跑 Claude Code 终端。手机上有个配套 App **KKCODER Mobile** 可以远程操控它。

我的电脑没有公网 IP，需要这台服务器做中转。

## 架构

```
手机 App → 这台服务器(frps) ===frp隧道=== 我电脑(KKCODER)
```

- 这台服务器跑 **frps**（frp 服务端）
- 我电脑跑 KKCODER + **frpc**（frp 客户端），frpc 连上来后把服务器的 9527 端口映射到我电脑
- 手机通过这台服务器访问，流量经 frps 隧道转到我电脑
- 同一时间只有一台电脑连接

## 连接方式（二选一）

### 方式一：IP 直连（简单快速）

手机直接访问 `服务器IP:9527`，走 frp 隧道到我电脑。

需要做的事：
1. 安装 frps，配置 token 认证，代理端口 9527
2. 开放防火墙 7000（frp 控制端口）和 9527（代理端口）
3. 设为系统服务开机自启

手机端填：IP 地址、端口 9527、HTTPS 关闭。

frp 本身有 token 认证，安全性没问题。

### 方式二：域名 + HTTPS（推荐）

手机访问 `https://域名`，nginx 做 SSL 终止后反代到 frps。

需要做的事：
1. 安装 frps，配置 token 认证，代理端口 9527
2. 安装 nginx，用 certbot 申请 SSL 证书，HTTPS 反代到 `127.0.0.1:9527`，支持 WebSocket
3. 证书自动续期加入 crontab
4. 开放防火墙 7000、9527、80、443
5. 设为系统服务开机自启

手机端填：域名、端口 443、HTTPS 开启。

---

两种方式 frps 配置一样，区别只是有没有 nginx 那一层。部署完请输出：frp Token、访问地址（IP:端口 或 域名）、Dashboard 地址。
